# Time-MoE Results

This directory stores benchmark artifacts generated by `models/time_moe_usage/run_benchmark.py`.

- `*_smoke/`: local CPU smoke-test runs.
- `*_a100/`: formal runs intended for the A100 8GPU server.
- Each run should contain `config.json`, `metrics.json`, `predictions.csv`, `summary.md`, and optionally `plot.png`.
